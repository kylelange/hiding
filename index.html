Flashcards!

Javascript(JS)

is a coding language that allows a website to interact with the user and compute information the user provides.

Operators

are used to perform arithmetic on numbers (+, -, *, /, %, ++, --).

Variables

are containers to store one of the five Data Types: strings, numbers, boolean, unidentified, and null.

Variables naming conventions

is a rule in JavaScript to always use camelCase for identifier names (variables and functions).

Functions

a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

Method

are the actions that can be performed on objects. Methods are functions stored as object properties.
